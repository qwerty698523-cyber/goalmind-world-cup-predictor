# GoalMind 世界杯预测网站

这是一个 2026 世界杯预测网站 Beta 版，包含实时赛程同步、球队动态战力、赛前预测、赛后复盘、淘汰赛模拟、天气与伤病变量、模型评价指标和发布前检查。

当前版本适合作为公开测试版发布：页面、接口和数据链路可以运行，模型评价系统已经接入；但模型质量还没有达到专业合格线，尤其是概率校准、总进球误差和样本外测试规模仍需继续迭代。

## 本地运行

双击：

```text
启动网站.cmd
```

或在命令行运行：

```cmd
cd /d C:\Users\gale\Documents\世界杯
start-site.cmd
```

默认地址：

```text
http://127.0.0.1:3000
```

## 发布前检查

```cmd
npm run release:check
```

这个命令会检查语法和测试、刷新赛程缓存、生成离线兜底数据，并输出 `release-report.json`。当前发布状态为 `publishable-beta`，表示可以作为测试版发布，但不代表模型已经达到正式专业质量门槛。

## 已实现功能

- 2026 世界杯赛程、比分、状态和球队数据同步；
- 48 支球队动态战力；
- 基于 Elo、近期状态、进攻、防守、比赛质量、天气、伤病和阵容可用性的赛前预测；
- 90 分钟常规时间胜、平、负概率；
- 最可能比分、前三候选比分、双方预期进球、总进球和双方进球概率；
- 淘汰赛区分“90 分钟赛果概率”和“最终晋级概率”；
- 若淘汰赛 90 分钟打平，页面会说明加时赛和点球大战路径；
- 完赛后显示最终比分旁的赛前预测；
- 预测结果颜色标记：比分和胜负都正确为绿色，只猜对胜负为黄色，方向错误为红色；
- 单场比赛页包含赛前预测、赛后总结、预测复盘和模型学习信息；
- 球队档案包含名单、教练组、首发/替补、阵型、人员结构、有球阶段和无球阶段分析；
- 小组出线模拟和淘汰赛路径模拟；
- 模型评价指标：命中率、Log Loss、Brier Score、ECE、平局召回率、宏平均 F1、精确比分命中率、前三比分覆盖率、进球误差等；
- 基准模型对比：随机基准、固定比分基准、基础 Elo、Elo+Poisson。

## 常用命令

```cmd
npm run check
npm run model:test
npm run model:replay
npm run model:gate
npm run release:check
```

其中：

- `check`：检查 JavaScript 语法；
- `model:test`：运行模型评价相关单元测试；
- `model:replay`：用当前模型重放已完赛比赛；
- `model:gate`：按正式模型质量门槛检查，当前不保证通过；
- `release:check`：发布前检查，允许作为测试版发布并记录未达标指标。

## 数据来源

- 实时赛程和比赛详情：ESPN 公开接口；
- 历史国家队比赛：International football results 数据集；
- 战力和预测：GoalMind 本地模型；
- 天气：按比赛场馆位置读取小时级天气；
- 阵容和伤病：优先使用公开名单与 ESPN roster 信息，缺失时回退本地估计。

预测仅供参考，不应使用“稳赢”“必胜”“百分百准确”等误导性表述。
