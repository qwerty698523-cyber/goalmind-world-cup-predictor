# GoalMind 世界杯预测

GoalMind 是一个面向球迷的世界杯观赛分析网站。它会把赛程、球队状态、Elo 战力、近期表现、阵容可用性、天气和伤病等信息汇总到一起，帮助你更快理解一场比赛可能会怎么发展。

你可以用它查看每场比赛的胜平负概率、候选比分、双方预期进球、赛前战术倾向、赛后复盘，以及小组出线和淘汰赛晋级路径模拟。

## 你可以看到什么

- **比赛预测**：展示 90 分钟常规时间的主胜、平局、客胜概率；
- **比分参考**：给出最可能比分，以及至少三个候选比分；
- **进球趋势**：展示双方预期进球、总进球倾向和双方是否都有进球的可能性；
- **淘汰赛说明**：区分“90 分钟赛果概率”和“最终晋级概率”。如果 90 分钟打平，比赛可能进入加时赛和点球大战；
- **球队档案**：查看球队名单、首发/替补、教练组、阵型、有球阶段和无球阶段特点；
- **赛后复盘**：完赛后对比赛前预测与真实比赛，标出预测命中、偏差和需要修正的地方；
- **出线路径**：模拟小组排名、晋级路径和后续对阵变化。

## 如何理解预测

预测不是“保证结果”，而是对比赛不确定性的量化表达。

例如，某队胜率为 60%，并不代表它一定会赢，而是表示在大量相似条件的比赛中，它更可能占优。足球比赛会受到红牌、点球、伤病、临场战术、天气和偶然进球影响，因此实际比分可能与预测不同。

页面中的颜色标记用于帮助你快速回看预测效果：

- 绿色：胜负方向和比分都正确；
- 黄色：胜负方向正确，但比分有偏差；
- 红色：胜负方向判断错误。

## 数据更新

网站会持续同步公开赛程、比分和球队信息。页面会显示最近更新时间；当比赛结束、阵容公布或赛程变化时，预测和模拟结果也会随之更新。

如果短时间内数据没有刷新，通常是公开数据源延迟或网络缓存造成的，稍后重新打开页面即可。

## 使用提醒

GoalMind 适合用于观赛前分析、赛程追踪和赛后讨论。预测结果只作为参考，不构成比赛保证，也不应被理解为投注建议。
